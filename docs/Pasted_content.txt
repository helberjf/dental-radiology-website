Crie um site de loja virtual para um laboratório odontológico digital, inspirado na estrutura da página de produto de referência enviada, mas sem copiar marca, textos, imagens ou identidade visual protegida.

Stack:
- Next.js com App Router
- TypeScript
- Tailwind CSS
- Componentes reutilizáveis
- Não usar React Router
- Layout responsivo para desktop e mobile

Objetivo:
Criar uma loja de serviços odontológicos digitais onde dentistas possam comprar ou solicitar serviços como conversão de arquivos, DICOM/STL, impressão 3D, biomodelos, cirurgia guiada, ortodontia, prótese e radiologia digital.
